Chrome DevTools MCP Server

本节介绍 DevTools MCP 的核心概念与 Workshop 中的实践要点。

什么是 MCP Server?

MCP (Modular Chrome Proxy / Controller 平台) —— 在 Workshop 场景中，指基于 Chrome DevTools Protocol 的轻量服务/代理，用于远程控制、自动化测试与性能诊断。

Workshop 覆盖内容

  1. 理解 DevTools 协议（CDP）的核心域与消息格式。
  2. 搭建本地 MCP Server，接入 DevTools 客户端。
  3. 演示网络、性能与 DOM 调试的自动化脚本。

建议参考 DevTools 官方文档、Workshop 示例仓库与社区示例代码。
